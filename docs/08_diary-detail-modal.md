# 08: 日記詳細モーダル

## 概要

日記カードをタップした際に表示される詳細画面。モーダルプレゼンテーションで表示。

## 対応内容

### 画面実装

- [ ] `app/diary/[id].tsx` 作成（動的ルート）
- [ ] `presentation: 'modal'` でスタックに登録
- [ ] 日付 + 生後〇日表示
- [ ] 写真フルワイド表示
- [ ] テキスト全文表示
- [ ] 著者情報（誰が書いたか）

### 操作

- [ ] 編集ボタン → 書く画面に既存データを渡して遷移
- [ ] 削除ボタン → 確認ダイアログ → 削除処理（自分の日記のみ）
- [ ] 削除後にホーム画面へ戻る

### 編集機能

- [ ] 書く画面を編集モードで再利用（既存の text / photo を初期値にセット）
- [ ] 更新処理（UPDATE + updated_at 更新）
