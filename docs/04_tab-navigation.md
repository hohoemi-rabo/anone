# 04: タブナビゲーション構成

## 概要

Phase 1 の画面構成に合わせてボトムタブナビゲーションを再構築する。

## 背景

現在のテンプレートは Home / Explore の2タブ構成。REQUIREMENTS.md の仕様に合わせて4タブ（ホーム / 書く / 思い出 / 設定）に変更する。

## 対応内容

### タブ構成変更

- [ ] `app/(tabs)/_layout.tsx` を4タブ構成に変更
- [ ] タブ1: ホーム（日記一覧）- アイコン選定・設定
- [ ] タブ2: 書く（新規作成）- アイコン選定・設定
- [ ] タブ3: 思い出（振り返り）- アイコン選定・設定
- [ ] タブ4: 設定 - アイコン選定・設定

### 画面ファイル作成

- [ ] `app/(tabs)/index.tsx` → ホーム画面（プレースホルダー）
- [ ] `app/(tabs)/write.tsx` → 書く画面（プレースホルダー）
- [ ] `app/(tabs)/memories.tsx` → 思い出画面（プレースホルダー）
- [ ] `app/(tabs)/settings.tsx` → 設定画面（プレースホルダー）
- [ ] `app/(tabs)/explore.tsx` 削除

### ヘッダー

- [ ] 子どもの名前 + 生後〇日 / 〇歳〇ヶ月をヘッダーに常時表示（共通コンポーネント）
- [ ] 生後日数の計算ユーティリティ作成
