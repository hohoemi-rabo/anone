# 11: 家族共有機能（招待コード）

## 概要

招待コード方式で1つの子どもプロフィールを複数ユーザーで共有する機能。

## 対応内容

### 招待コード発行（owner のみ）

- [ ] 設定画面から「家族を招待」→ 招待コード発行画面
- [ ] 6桁英数字コードを生成
- [ ] `invite_codes` テーブルに INSERT（expires_at = 24時間後）
- [ ] コードのコピー機能
- [ ] OS シェアシート連携（`Share` API）

### 招待コード参加

- [ ] 設定画面から「招待コードで参加」→ コード入力画面
- [ ] 6桁コード入力 UI
- [ ] コード検証（存在確認、有効期限チェック、使用済みチェック）
- [ ] 有効なコードの場合、`child_members` に `role: 'member'` で追加
- [ ] `invite_codes.used = true` に更新
- [ ] 参加完了後、ホーム画面へ遷移

### 権限制御

- [ ] 日記一覧で「誰が書いたか」をアバター or イニシャルで表示
- [ ] 日記削除は自分の日記のみ可能（UI 上も他人の日記は削除ボタン非表示）
- [ ] 子ども情報変更は owner のみ（設定画面で UI 制御）
- [ ] メンバー管理画面（owner のみ表示、メンバー一覧 + 削除機能）

### 制約

- [ ] 最大5人まで共有可能（超過時にエラーメッセージ）
- [ ] 招待コード有効期限: 24時間
