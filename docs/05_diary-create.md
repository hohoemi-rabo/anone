# 05: 日記作成画面（書く）

## 概要

今日の日記を作成する画面。アプリのコア機能。

## 背景

「開いてすぐ書ける」が設計原則。起動後2タップ以内で記録開始できることが目標。

## 対応内容

### 画面実装

- [ ] `app/(tabs)/write.tsx` 実装
- [ ] 日付表示（今日の日付、変更不可）
- [ ] 生後〇日表示
- [ ] テキスト入力エリア（プレースホルダー: 「今日はどんな日でしたか?」、最大500文字）
- [ ] 文字数カウンター表示
- [ ] 写真追加ボタン（1枚のみ）
- [ ] 保存ボタン

### キーボード対応

- [ ] キーボード表示時に写真ボタン・保存ボタンがキーボード上に追従
- [ ] `KeyboardAvoidingView` または同等の実装

### 保存処理

- [ ] `diary_entries` テーブルに INSERT
- [ ] 同日の日記が存在する場合は「今日の日記を編集しますか?」と確認ダイアログ
- [ ] 保存成功時にホーム画面へ遷移（静かなアニメーション）

### 写真添付

- [ ] カメラロールから写真を1枚選択（`expo-image-picker`）
- [ ] 選択後プレビュー表示・削除可能
- [ ] 保存時に画像圧縮 + Supabase Storage アップロード（詳細は 07_image-handling.md）
