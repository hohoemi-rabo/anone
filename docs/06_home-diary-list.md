# 06: ホーム画面（日記一覧）

## 概要

書いた日記を時系列で確認するメイン画面。

## 対応内容

### 日記一覧

- [ ] `app/(tabs)/index.tsx` 実装
- [ ] `diary_entries` を新しい順に取得・表示（FlatList / FlashList）
- [ ] 日記カード コンポーネント作成
  - 日付（例: 4月15日 火曜日）
  - 生後〇日表示
  - テキスト（最大3行で省略）
  - サムネイル画像（写真がある場合）
  - 誰が書いたか（アバター or イニシャル）

### 1年前の今日

- [ ] 1年前の同日の日記を取得
- [ ] 該当がある場合、画面上部にカード表示
- [ ] 該当なしの場合は非表示（セクション自体を出さない）

### インタラクション

- [ ] カードタップ → 日記詳細モーダルへ遷移
- [ ] 長押し → 削除確認ダイアログ（自分の日記のみ）
- [ ] 書くタブへの誘導ボタン（FAB or 右上ボタン）

### データ取得

- [ ] ページネーション対応（無限スクロール）
- [ ] プルトゥリフレッシュ
- [ ] 空状態の表示（まだ日記がない場合の案内）
