# 09: 思い出画面（振り返り）

## 概要

過去の日記を感情的に楽しむための振り返り画面。3つのセクションで構成。

## 対応内容

### 1年前の今日セクション

- [ ] 1年前の同日の日記を取得・表示
- [ ] 写真 + テキスト + 当時の生後日数
- [ ] 該当なしの場合はセクション非表示

### 成長タイムライン

- [ ] 月別に区切られた日記リスト
- [ ] 「生後〇日〜〇日」でグルーピング
- [ ] 写真付き日記は画像を強調表示
- [ ] セクションリスト（SectionList）で実装

### カレンダービュー

- [ ] 月カレンダー形式の表示
- [ ] 日記を書いた日にのみドット表示
- [ ] 書かなかった日に x マークや強調は一切しない（設計原則）
- [ ] 日付タップで当日の日記詳細モーダルへ遷移
- [ ] カレンダーライブラリの選定（`react-native-calendars` 等）

### セクション切り替え

- [ ] 3セクションの切り替え UI（タブ or セグメンテッドコントロール）
